## Requirements

- Node JS
- npm
- [Create Open AI account](https://beta.openai.com/signup/)

## Setup

1. Clone the repository

```
git clone https://github.com/satakshigarg/ChatGPT.git
```

2. Install client dependencies

```
cd ChatGPT
npm install
```

## Usage

1. Start the app

```
npm run dev
```

2. Start the app using run.sh

```
chmod +x run.sh
./run.sh

```
